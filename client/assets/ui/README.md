# UI assets (replaceable PNG placeholders)

Put your own PNG icons in this folder. The game is already wired to these names.

## Current files used
- `coin.png` → HUD + menu coin icon

## Recommended export sizes
- `coin.png`: **128x128 px** (square, transparent background)

## Optional files for later
- `logo.png`: 1024x512 px
- `skins-button.png`: 256x256 px
- `leaderboard-badge.png`: 128x128 px

If `coin.png` is missing, the UI falls back to an emoji coin automatically.
