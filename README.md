
# tralala.io starter

A polished full-stack starter for a Paper-style territory game.

## Features
- Guest join flow
- Skin picker with 3 starter skins and 3 locked placeholders
- Multiplayer room with Socket.IO
- Bot players
- Territory claiming
- Trail elimination
- Coins from kills
- Leaderboard
- White chat bubbles
- Mobile joystick
- Editable asset folders

## Run
```bash
npm install
npm start
```

Open:
```text
http://localhost:3000
```

## Asset folders
- `public/assets/skins/`
- `public/assets/gallery/`
- `public/assets/audio/`

This starter is intentionally structured so you can replace placeholders with real art, UI assets, gallery content and music later.
