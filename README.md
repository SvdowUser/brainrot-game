# Los Tralaleritos 3D Hub – Working Guest Starter

This version avoids the form-submit reload bug by not using a form for the guest entry at all.

## What works
- Guest entry with local username storage
- Enter key works
- Click button works
- 3D beach hub in browser
- Local chat panel
- Local speech bubble above the player
- Mobile joystick
- Placeholder roaming guests

## Important
This is a frontend-only guest lobby for GitHub Pages.
Real multiplayer, voice chat, unique usernames, rooms and persistence need a separate backend later.
