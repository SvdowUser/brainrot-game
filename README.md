# Los Tralaleritos Beach Hub

Fullscreen English top-down hub starter for GitHub Pages.

## Included
- beach starter map
- free movement hub
- mobile touch joystick
- keyboard controls
- camera follow
- locked future areas with construction barriers
- placeholder portal for mini games

## Controls
- Desktop: WASD or Arrow Keys
- Mobile: touch on the left side for joystick
